## Using Variables in MDX Files

You can utilize variables in your MDX files by following these steps:

1. Create a JavaScript or TypeScript file to define your variables. For example, let's call it `data.js` or `data.ts`.

2. Inside the file, use the `export` keyword to define your variables. For example:

```jsx copy filename="data.ts"
export const person = {
  age: 24,
  name: "Foo",
  profession: "Bar",
};

export const a = 1;
```

3. In your MDX file, import the variables using the import statement. For example:

```mdx copy filename="sample.mdx"
import { person, a } from "./data";
```

4. Once imported, you can access the variables using curly braces `{}` in your MDX file. For example:

```mdx copy filename="sample.mdx"
The person's age is {person.age}
The value of 'a' is {a}

```

## Example

```ts copy filename="data.ts"
export const  person  = {
    age: 24,
    name: "Foo",
    profession: "Bar"
}
```
